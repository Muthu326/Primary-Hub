# How to Launch "Primary Learn Hub" as a Permanent Website 🌐

Follow these 3 steps to get your school's app online permanently for free:

### 1. Upload to GitHub
1.  Create a free account on [GitHub.com](https://github.com).
2.  Create a new repository called **`Primary-Learn-Hub`**.
3.  Upload all files from your folder (`app.py`, `modules/`, `utils/`, `assets/`, `requirements.txt`, etc.) to this repository.

### 2. Connect to Streamlit Cloud
1.  Go to [share.streamlit.io](https://share.streamlit.io) and log in with your GitHub account.
2.  Click **"New app"**.
3.  Select your repository (`Primary-Learn-Hub`), the branch (`main`), and the main file (`app.py`).
4.  Click **"Deploy!"**.

### 3. Your Live Website is Ready! 🚀
- In a few minutes, you will get a permanent URL like:
  `https://primary-learn-hub.streamlit.app`
- You can share this link with all students, parents, and teachers!

---

> [!IMPORTANT]
> **Data Persistence**: Streamlit Cloud is free but it resets your database (`school.db`) if the app is inactive for a long time. For the 100th-anniversary event, we recommend having a teacher keep the website open on one laptop to keep it "active".
